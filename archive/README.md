# Archive Directory

This directory contains archived files that are no longer actively used but preserved for historical reference.

## Directory Structure

```
archive/
├── README.md                          # This file
├── MIGRATION_SUMMARY.md               # Image migration summary
├── Squarespace-Wordpress-Export-11-18-2025.xml  # Original Squarespace export
├── pages/                             # Archived HTML pages
│   ├── README.md                      # Pages documentation
│   └── [9 archived HTML pages]
└── scripts/                           # Archived Python scripts
    └── [legacy migration scripts]
```

## Contents

### Export Files
- `Squarespace-Wordpress-Export-11-18-2025.xml` - Original Squarespace/WordPress export file used during migration (November 18, 2025). This file contains all the original content, posts, and metadata from the Squarespace site.

### Documentation
- `MIGRATION_SUMMARY.md` - Summary of the image migration process from Squarespace CDN to local hosting
- `pages/README.md` - Documentation for archived HTML pages
- `scripts/` - Legacy Python scripts used during migration

### Archived Pages
See `pages/README.md` for details on archived HTML pages.

## Notes

- All files in this directory are preserved for historical reference
- Files may be referenced in documentation or used for future migrations
- The XML export file can be used to restore original content if needed
- Pages can be restored to the root directory if needed
